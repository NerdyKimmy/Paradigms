# Haskell and Prolog Programming Paradigm Labs

## Prerequisites

- Install `ghc` Haskell compiler from the [official website](https://www.haskell.org/ghc/download) or using the package manager example `slpkg install ghc`

## Compiling and launch

- Choose the directory with the problem-solving you want. 
- Run compilation `ghc main.hs`
- Launch and be happy :3 `./main`
